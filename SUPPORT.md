## Questions & Issues

- **Bug reports** — Open an issue using the [bug report template](.github/ISSUE_TEMPLATE/bug_report.md)
- **Feature requests** — Open an issue using the [feature request template](.github/ISSUE_TEMPLATE/feature_request.md)
- **General questions** — Open a [GitHub Discussion](https://github.com/erfnzdeh/git-retime/discussions) or an issue

## Before Asking

1. Check existing [issues](https://github.com/erfnzdeh/git-retime/issues) and [discussions](https://github.com/erfnzdeh/git-retime/discussions)
2. Ensure you're on the latest release
3. Include your OS, Go version, and steps to reproduce (for bugs)
